# Política de Privacidad – Mis Tareas

**Última actualización:** Febrero 2026

La aplicación **Mis Tareas** respeta tu privacidad. Esta política explica de forma clara y sencilla cómo se manejan los datos dentro de la aplicación.

---

## 1. Información que recopilamos

**Mis Tareas no recopila información personal del usuario.**

La aplicación únicamente guarda la información que el usuario escribe manualmente (como tareas o notas) y esta se almacena **exclusivamente en el dispositivo del usuario**.

---

## 2. Uso de la información

Los datos guardados en la aplicación se utilizan únicamente para el funcionamiento de la app, permitiendo al usuario:

* Crear tareas
* Editar tareas
* Eliminar tareas

La información **no se envía a internet**, **no se comparte** y **no se vende a terceros**.

---

## 3. Servicios de terceros

La aplicación **no utiliza**:

* Servicios de publicidad
* Herramientas de analítica
* Sistemas de seguimiento
* Servicios en la nube

Todo funciona de forma local en el dispositivo.

---

## 4. Almacenamiento de datos

Los datos se guardan localmente usando almacenamiento interno del dispositivo.
Si el usuario desinstala la aplicación o borra sus datos, la información se elimina.

---

## 5. Permisos

La aplicación no solicita permisos sensibles ni accede a información personal del dispositivo.

---

## 6. Cambios a esta política

Podemos actualizar esta política si en el futuro la aplicación agrega nuevas funciones.
Cualquier cambio se publicará en esta misma página.
